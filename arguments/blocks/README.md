# Argument Blocks — Index

**Layer 1 of the knowledge bank.** Each file is one atomic argument = a warranted causal
chain + its clash (`They say → I say`), distilled from a *graded* practice speech. Blocks
link up to reusable [[mechanism-bank]] chains and [[examples-bank]] cases rather than
repeating them. Built and refiled by the `distill-arguments` skill.

**Filed by primary `#theme` tag.** A motion can split across folders on purpose — the
animal-research case lives in both `ethics/` (rights limb) and `science-environment/`
(practical limb). Reassemble any one motion with its provenance line, e.g.:
`grep -rl "used in \[\[day-11\]\]" .`

**Legend:** 🏆 personal-best landed · ✅ landed in-round · ◐ mixed / partial · ⚠️ anti-pattern
(kept as a caution, with the failure recorded) · 🔁 transferable rebuttal move (motion-agnostic shield)

## How to use this drawer
- **Prepping a similar motion?** Open the topic folder, pull the blocks, swap the specifics.
  The 🔁 blocks are motion-agnostic shields — reach for them on *any* motion of that shape.
- **Learning from what worked?** 🏆/✅ blocks landed in a real graded round — trust their moves.
  ⚠️ blocks are kept *because* they flopped; the `SOURCE` line records exactly why, so you
  don't repeat the mistake (false stats, self-concessions, caricatures).
- **Growing the bank:** run `/distill-arguments` after a practice day — it files new blocks
  into the right topic folder and enriches [[mechanism-bank]] / [[examples-bank]].
