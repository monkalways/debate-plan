ARGUMENT: banning forces in-person interaction → healthy development
Side: Prop        Motion type: Policy
Burden it serves: second, independent substantive (a separate engine, not mental health rebadged)

CLAIM: Banning social media forces children into face-to-face interaction, which is necessary for healthy social development.

CHAIN:
  L1  Digital alternatives satisfy the need for connection from home → kids stop meeting in person
      Warrant: convenience of Discord/Instagram substitutes for physical hangouts; friend-group logic — if only ~1/3 want to meet in person, the 2/3 majority keeps everyone inside.
      Evidence: [[examples-bank#Teens prefer face-to-face]] — ~1/3 of teens prefer in-person communication (Time/Common Sense)
  L2  No in-person practice → social-development skills atrophy/halt
      Warrant: empathy, honesty, reading people are built only through real interaction; kids used to online convenience never learn to set up in-person contact.
      Evidence: —
  L3  Ban removes the digital hatch → the non-negotiable need for connection drives them outside → development resumes
      Warrant: connection can no longer be met on a screen, so kids meet physically, and face-to-face demands the skills.

IMPACT: a generation develops social competence in its formative years.
  Weighing: timeframe + scope — "we achieve this in a much shorter time frame" (7:24), across a whole cohort's formative years; opp only concedes kids should "eventually" go outside.

THEY SAY → I SAY:
  They say: social media builds connection.  → I say: ~80% of time is passive consumption, only ~20% active engagement; real connection is face-to-face, so theirs is thin. (⚠️ the 80/20 "SQ Magazine" split is unverified — use as reasoning, not a hard stat)
  They say: age-verification is privacy-invasive (Yoodli follow-up).  → I say: Australia/UK already run it — more age checks on sites, not mass surveillance; friction is enough for a marginal net benefit.
