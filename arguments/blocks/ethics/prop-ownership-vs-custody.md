ARGUMENT: returning ownership ≠ moving the object forever — the origin nation can own it and loan it back  🔁
Side: Prop        Motion type: Value/Policy   (reframe device — defuses the preservation objection WITHOUT conceding ownership)
Burden it serves: separates the legal question (who owns it) from the practical one (where it's stored), so preservation fears can't block return.

CLAIM: Ownership and custody are two different things — the origin nation can hold legal ownership while the object physically stays in a secure museum on loan, so "return" needn't mean the object ever leaves a climate-controlled gallery.

CHAIN:
  L1  Return legal ownership → the origin nation decides where the object lives
      Warrant: owning a thing means controlling it, not necessarily housing it; owners lend possessions all the time.
  L2  Owner can choose to loan it back → the object can stay in the current museum
      Warrant: if the origin nation judges a facility safer, it can leave the object there on its own terms — custody without surrendering title.
  L3  Ownership settled + custody flexible → justice AND preservation both met
      Warrant: the wrong (who owns it) is corrected while the object stays exactly as protected as opp wants.

IMPACT: preservation stops being a reason to refuse return — the object can be owned by the origin nation and safe in the same building.
  Weighing: this dissolves the clash rather than trading it off — opp's preservation win costs Prop nothing, because ownership can return without the object moving.

THEY SAY → I SAY:
  They say: many origin countries lack climate-controlled facilities, so return risks the object.  → I say: return the OWNERSHIP, not necessarily the object — the origin nation can loan it straight back to a secure museum; they own it, they just also choose to keep it protected.
  They say: then nothing really changes.  → I say: everything that matters changes — the world formally acknowledges whose it is, and THEY decide its future instead of the people who took it.
