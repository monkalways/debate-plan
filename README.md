# debate-plan

A structured summer training program (July–August 2026) to develop **Lucas's** World
Schools debate skills, targeting his two goals: **prep a 6-minute speech in 15 minutes**,
and build a reusable **library of motions, arguments, and mechanisms**.

## Start here
👉 **[`START-HERE.md`](START-HERE.md)** — the launchpad (rules, Day-1 runsheet, file map).

Then: **[`DAY-1.md`](DAY-1.md)** → **[`WEEK-1.md`](WEEK-1.md)** → the weekly template in
[`plans/summer-2026.md`](plans/summer-2026.md).

## What's inside
| Path | Purpose |
|------|---------|
| `profile.md` | Lucas's background, achievements, strengths/weaknesses, goals |
| `plans/summer-2026.md` | The full 8-week training plan |
| `plans/progress-log.md` | Rubric + log to track skill (not busywork) |
| `motions/motion-taxonomy.md` | The 8 motion types — burden + how to approach each |
| `motions/motion-list.md` | ~60 motions tagged by type/domain/difficulty + drill sets |
| `motions/worked-cases.md` | Full model cases in the 4-link chain format |
| `arguments/mechanism-bank.md` | Pre-built argument chains + weakest-link / "they say → I say" |
| `arguments/examples-bank.md` | Real examples + stats to cite |
| `reference/framing-and-style.md` | Intros, signposting, rhetoric — the style toolkit |
| `.claude/agents/debate-coach.md` | A reusable "veteran coach" agent for reviewing materials |

## How it was built
The plan and materials were drafted and then critiqued by a simulated veteran World
Schools coach agent, with fixes applied across several review passes.
