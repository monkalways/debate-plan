# START HERE — Lucas's Summer Debate Kit

Your one-page launchpad. Read this first; it points you to everything else.

## The 3 rules (the whole philosophy)
1. **The clock is a thermometer, not the furnace.** Prep gets faster *because* your
   library is automatic — don't chase the minutes. If hitting the clock makes you drop
   weighing, the rep failed. Slow down.
2. **Weighing is how you win.** Always end an argument with *"and we outweigh because…"*
   (scope / probability / magnitude / reversibility / timeframe).
3. **Scaffolds, not scripts.** The banks are structures to *adapt*, never lines to recite.
   Memorised = you've used them wrong.

## Day 1 — your first 45 minutes
1. **Read** `motions/motion-taxonomy.md` (the 8 motion types). ~15 min.
2. **Type Drill:** open `motions/motion-list.md` → do **Set A**. Say each motion's *type*
   + *the one thing you must prove*, on a timer. ~5 min.
3. **Baseline speed test:**
   - Pick one 🟢 **Economics** motion from `motion-list.md` (e.g. "THW tax sugary drinks").
   - Prep **30 minutes** using the protocol in `plans/summer-2026.md` (define → 3 chains →
     examples → **protected 90-sec weighing pass** → intro).
   - **Record** the 6-minute speech.
4. **Score it** honestly in `plans/progress-log.md` → "Week 1 — Baseline" box.
   That's your starting line. Everything is measured against it.

## The daily loop (~1.75 hrs, 6 days, Sunday OFF)
- **Block A — Knowledge (30 min):** read the week's domain → notes → file an example.
- **Block B — Skill drill (60 min):** rotates (see weekly template in the plan).
- **Block C — Micro-drills (20 min):** 90-sec weighing drill + POI drill + record/style.

## The weekly rhythm
Mon speed-write · Tue live-clash · Wed build library · Thu speed-write · **Fri coach** ·
Sat style + **peer recording-swap** + record · **Sun OFF**.
Full detail: `plans/summer-2026.md`.
