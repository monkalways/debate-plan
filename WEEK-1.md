# Week 1 — Day-by-Day (Economics & Development)

Week 1 domain: **Economics & Development.** Speed clock: **30 min** (it's a thermometer —
don't sacrifice weighing to beat it).

Each day = **Block A** (Knowledge ~30 min) + **Block B** (the day's main drill ~60 min) +
**Block C** (Micro-drills ~20 min). Only Block B changes day to day.

- **Monday (Day 1):** onboarding + baseline test — see **`DAY-1.md`**.
- **Sunday:** OFF (genuine rest — no logging, no reading).

> Every day, Block C = **90-sec weighing drill + POI drill + a short style/recording rep.**
> It's the same shape daily; only the focus shifts slightly. Listed per day below.

---

## Tuesday (Day 2) — Live-clash / rebuttal

**Block A — Knowledge (30 min)**
- Read about **one** Economics sub-theme (pick from `examples-bank.md`: inequality,
  market regulation, trade, or automation). Read 2–3 short sources.
- Add **one fresh example with a stat** to `arguments/examples-bank.md` (tag it; mark ⚠️ if
  the number needs verifying).

**Block B — Live-clash drill (60 min)**
- Take **yesterday's sugary-drinks Prop case**.
- Have your **AI sparring partner or a peer** fire **3 opposing arguments** at it, one at a time.
- Respond to each out loud using the move: *"They say X →"* then pick one:
  **(a) it's wrong**, **(b) it doesn't matter** (you outweigh), or **(c) it's true but irrelevant.**
- Use the **weakest-link tags** and **"they say → I say"** rows in `mechanism-bank.md` for ammo.
- Do 2–3 rounds. The goal is *responsiveness*, not a perfect script.

**Block C — Micro-drills (20 min)**
- 90-sec weighing drill (on the sugary-drinks case) · POI drill (take 2 mid-flow) ·
  2-min impromptu on any Economics motion.

---

## Wednesday (Day 3) — Build argument blocks (library day)

**Block A — Knowledge (30 min)**
- **Deep-dive one issue** (e.g. "does redistribution hurt growth?" — the efficiency–equity
  tradeoff). Read enough to argue *both* sides.

**Block B — Build the library (60 min)**
- Build **4-link chains for 2 new Economics motions** from `motion-list.md`
  (suggested: *THW cancel the debt of developing nations* and *THBT free trade has done
  more good than harm*).
- For each: 3 Prop + 3 Opp arguments, pulling mechanisms from the bank.
- **Add a "weakest link" tag** to each new chain (the link an opponent attacks).
- File them in `motions/worked-cases.md` (update the index at the top).

**Block C — Micro-drills (20 min)**
- Signposting drill (deliver one chain with crisp "first… second…" signposting) ·
  90-sec weighing drill.

---

## Thursday (Day 4) — Speed case-write #2

**Block A — Knowledge (30 min)**
- Read Economics source #3 → notes → file an example.

**Block B — Speed case-write #2 (60 min incl. delivery)**
- **Motion:** *THW implement a universal basic income* — argue **Opposition** (contrast the
  side you've been on).
- **30-min prep** using the protocol (define → 3 chains → examples → protected 90-sec
  weighing pass → intro). Clock is 30 min.
- **Record** the 6-min speech.
- **Then self-check:** compare your case to the model in `worked-cases.md` **Motion 4 (UBI)**.
  What did the model do that you missed? (This is built-in feedback — use it.)

**Block C — Micro-drills (20 min)**
- Score the recording on the rubric → log it (full weekly log row) · POI drill.

---

## Friday (Day 5) — FDT coach session

**Block A — Knowledge (30 min)**
- Review the week's notes. Jot **2–3 questions** to ask the coach — aim them at your
  weak spots (weighing, clash), not just "was that good?"

**Block B — Coach session (60 min)**
- Mock round / POIs / feedback with the FDT coach.
- **Bring** your week's recordings and the questions. This is your main *substance*
  feedback — get human eyes on **weighing and clash** specifically.

**Block C — (in session)**
- Capture the coach's feedback in the log as a "one fix" for next week.

---

## Saturday (Day 6) — Style + peer swap + record

**Block A — light (examples already captured during the week)**
- Optional: tidy the week's examples in `examples-bank.md`.

**Block B — Style + peer swap (60 min)**
- **Style rewrite:** take one argument from this week and rewrite it vividly using
  `reference/framing-and-style.md` (show-don't-tell + one device). Read both versions
  aloud — hear the difference.
- **Peer recording-swap:** you watch a teammate's speech, they watch yours; **each offers
  2 POIs.** Trade one concrete fix.

**Block C — Record + close the week (20 min)**
- Record a full **6-min speech** on any Economics motion; score it on the rubric → log.
- **End-of-Week-1 check:** compare today's scores to **Monday's baseline.** Biggest gain?
  Still stuck? Write it down — it sets Week 2's focus.
